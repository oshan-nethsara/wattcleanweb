# Watt Clean Pvt Ltd - Solar Panel Cleaning Website

## Overview

This is the official website for **Watt Clean Pvt Ltd**, a professional solar panel cleaning and preventive maintenance company based in Sri Lanka. The website showcases our services, projects, and provides a contact form for potential clients.

## Features

- **Responsive Design**: Optimized for desktop and mobile devices
- **Navigation**: Smooth scrolling navigation to different sections
- **Services Showcase**: Detailed list of our core maintenance services
- **Project Gallery**: Links to completed projects
- **Contact Form**: Direct inquiry submission
- **WhatsApp Integration**: Quick contact via WhatsApp
- **SEO Optimized**: Meta tags for better search engine visibility

## Services Offered

- Solar Panel Cleaning
- Inverter Cleaning & Voltage Check
- DC & AC Cable Inspection
- Surge Protection Testing
- Grounding System Check

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Custom styling with responsive design
- **JavaScript**: Interactive elements and form handling
- **Font Awesome**: Icons for social media and UI elements
- **CDN**: Font Awesome library for icons

## How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/oshan-nethsara/wattcleanweb.git
   cd wattcleanweb
   ```

2. Open `index.html` in your web browser.

No server setup required - this is a static website.
